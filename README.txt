GODIUS 録画・YouTube配信用バッチ 

-- 制作日: 2024-10-01
-- 制作者: 鉄腕なんこつ

--------------------------------------

■概要■

このバッチ群は、オンラインRPG GODIUSのプレイ画面をローカルに録画 または YouTube上にライブ配信するための環境です。

・動作確認：Windows7以降 + PowerShell

・必要なアプリケーション：Chocolatey, FFmpeg
　※インストーラーを同梱しています

--------------------------------------

■使用方法■

1. 同梱の"install.bat"を管理者モードで開き、ChocolateyとFFmpegをウィザードに従ってインストールしてください。

2. 同じく同梱の"config.bat"を開き、下記の通り必要項目を入力してください。

　・動画の保存先：デフォルトでは、このバッチを展開したディレクトリとなっています。必要に応じてフルパスを指定してください。
　　※ 該当ディレクトリは必ず書き込み権限を許可しておくこと (読み込みのみになっていないこと)

　・YouTubeストリームキー：YouTubeで動画を配信する際に必要です。

3. 録画・配信：
　・Windowsキー+Rにて開かれたダイアログに"cmd"と入力すると、コマンドプロンプトが出てきます。
　　そこに"godius_live.bat"をドロップしエンターを押すと、ウィザードが起動します。
    ※バッチ起動後、下記ステップ5に進むまでに GODIUSのログイン準備をしておいてください。

　　1. マイク・サウンドミキサーのデバイスを選択してください。無音を指定することもできます。

　　2. BGMを選択できます。ネットワークストリームまたはローカルのメディアファイルを指定してください。無音を指定することもできます。

　　3. バッチの使用用途を選択してください。配信・録画・両方を選択できます。

　　4. GODIUSプレイ画面にて チャット欄を表示するかを選択できます。デフォルトでは非表示です。

　　5. 録画開始までの待機時間(秒)を選択できます。0?から指定できます。

　　6. 待機時間を過ぎると、自動的に録画が開始されます。

　　7. ※ 録画終了時は速やかにGODIUSからログアウトし、Ctrl+C押下→Yでバッチを終了してください。
